## Configration

- Enter token in .env file with `TOKEN`
```js
{
  "prefix": '!',
  "owner":["YOUR DISCORD ID"],
  "nodes": [{
    "name":"v1",
    "host":"localhost", // your lavalink host
    "port": 6969, // Your lavalink port
    "password": "iamnotdumb" // Your lavalink pass
  }]
}
```
### Change Status
- Go To `/events/ready.js`
